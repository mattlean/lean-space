Description

A Room Too Far is a dungeon crawler arcade game. The project is a proof of concept that explores the technical systems and play space dynamics between arcade machines, a web service, and its players.

The website hosts the level editor where players can build and create levels. The server then stores these levels and serves them to the arcade machine. Players then battle through these levels on the machine and their character data is stored back into the server, allowing for persistent character progression.
